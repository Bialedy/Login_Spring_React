Descrição
Projeto de exemplo com autenticação simples utilizando React no frontend e Spring Boot no backend.
A aplicação possui uma tela de login desenvolvida com React e Material UI (MUI). Após realizar o login, o usuário é redirecionado para um dashboard do sistema, onde é possível navegar pelo menu principal, 
acessar a página de alunos e utilizar a opção de sair (logout) do sistema.
O dashboard apresenta atalhos para diferentes áreas do sistema, como Alunos, Cursos e Professores, além de uma interface simples e organizada para navegação entre as funcionalidades.
Este projeto foi desenvolvido com fins de estudo para praticar a integração entre frontend e backend, além da construção de interfaces modernas utilizando React.

🛠 Tecnologias utilizadas

Frontend:

React
Material UI (MUI)
JavaScript
HTML / CSS

Backend:

Java
Spring Boot
Maven

Como executar o projeto
1. Clonar o repositório

2. Executar o backend
   
Abra a pasta backend 
Execute a classe principal do Spring Boot
Ou pelo terminal:
mvn spring-boot:run

O backend iniciará em:
http://localhost:8080

3. Executar o frontend
Abra a pasta frontend e execute:
npm install
npm start

O frontend iniciará em:
http://localhost:3000
